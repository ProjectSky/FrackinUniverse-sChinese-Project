###### 翻译格式
* 翻译源文件为Json格式，Eng表示原文，Chs表示中文。
* 未翻译源文件：
``` json
[
  {
    "DeniedAlternatives": [],
    "Files": {
      "npcs/crew/crewmemberbountyhunter.npctype": [
        "/scriptConfig/dialog/crewmember/offer/default/default/0"
      ]
    },
    "Texts": {
      "Eng": "Hello adventurer, mind if I join you?"
    }
  }
 ```
* 以上为一个未翻译源文件示例。

* 已翻译源文件：
``` json
[
  {
    "DeniedAlternatives": [],
    "Files": {
      "npcs/crew/crewmemberbountyhunter.npctype": [
        "/scriptConfig/dialog/crewmember/offer/default/default/0"
      ]
    },
    "Texts": {
      "Chs": "你好冒险家，介意我加入你们吗？",
      "Eng": "Hello adventurer, mind if I join you?"
    }
  }
 ```
* 以上为一个已翻译源文件示例，遵循Json格式即可。
